Gestor de projetos

O gerente de projetos é o profissional que desempenha o papel de gerenciar o desenvolvimento de um ou mais projetos estratégicos em uma empresa. Ele deve ser capaz de atingir os objetivos de cada um dos projetos por ele gerenciados conforme todas as áreas de conhecimento, processos e pessoas envolvidas.

Arquiteto de TI

O arquiteto de TI cuida da infraestrutura e garante que ela esteja condizente com as necessidades organizacionais. É importante citar que os elementos com os quais esse profissional trabalha são dispositivos móveis, computadores, itens de armazenamento e de impressão, redes de comunicação e dados.

Programador

O programador é o profissional que cria, desenvolve e mantém diferentes tipos de softwares em sistemas amplos ou para uso em computadores pessoais. Ele usa linguagens de programação, como html, java, c++, para criar comandos em programas, criando novas funcionalidades e utilidades.

Administrador de banco de dados

O que faz um DBA (Administrador de Banco de Dados)? O DBA (database administrator), sigla em inglês para Administrador de Banco de Dados, é um profissional da área de tecnologia responsável pela criação, instalação, monitoramento, reparos e análise de estruturas de um banco de dados.

Desenvolvedor web

Basicamente, o web developer é um tipo de programador, especializado em desenvolvimento de sistemas para a internet, ou seja, é a pessoa responsável por manter o seu site ou aplicativo em perfeito funcionamento.

Big data

Big Data é a análise e a interpretação de grandes volumes de dados de grande variedade. Para isso são necessárias soluções específicas para Big Data que permitam a profissionais de TI trabalhar com informações não-estruturadas a uma grande velocidade.
